Utiliser l'IA comme un vrai mentor

Les formules qui fonctionnent — et les erreurs à ne pas faire

✅ Les formules à copier

1Comprendre un concept : "Explique-moi [concept] comme si j'avais 12 ans, puis donne-moi un exemple concret en Python."
2Déboguer : "Voici mon code [code] et mon erreur [erreur]. Identifie le bug, explique pourquoi il se produit, et montre-moi la correction commentée."
3Se tester : "Je viens de terminer [sujet]. Pose-moi 5 questions pour valider ma compréhension. Attends mes réponses une par une."
4Démarrer un projet : "Je veux créer [projet]. Aide-moi à le décomposer en étapes, en commençant par la plus simple."
5Code review : "Voici mon code [code]. Joue le rôle d'un senior dev et dis-moi ce qui pourrait être amélioré, sans tout réécrire."
6Débloquer : "Je bloque depuis 2h sur [problème]. Aide-moi à le décomposer par questions, sans me donner la réponse directement."
7Reprendre après une pause : "Je reprends ma formation après [X jours]. Voici où j'en suis : [résumé]. Propose-moi un plan pour aujourd'hui."

⚠ Les erreurs à éviter

✗Ne pas demander : "Écris-moi tout le code du projet." → Tu ne vas rien apprendre. Demande des indices progressifs.
✗Ne pas copier-coller sans comprendre. Si tu ne peux pas expliquer le code ligne par ligne, c'est trop tôt.
✗Vérifier toujours les réponses techniques de l'IA avec la documentation officielle. L'IA peut se tromper.
